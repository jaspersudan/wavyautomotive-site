## 🐛 Troubleshooting

### Logo not showing
- Check file is in `assets/` folder
- Refresh page (Ctrl+F5)
- Check file name is `logo.svg`

### Admin password won't work
- Password is case-sensitive
- Default is `admin123` (all lowercase)
- Check you haven't changed it

### Inventory window won't open
- Check browser popup blocker
- Try opening with middle-mouse button
- Try different browser

### Data not saving
- Check browser allows cookies/storage
- Try clearing cache
- Check localStorage isn't disabled

See QUICKSTART.md for more troubleshooting.
