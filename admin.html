SharePoint Integration

Connect to SharePoint Excel Sheet

This feature allows you to sync your inventory with an Excel sheet stored on SharePoint. When enabled, your inventory will automatically update based on changes to the Excel file.

SharePoint Configuration

The spreadsheet should have columns: Make, Model, Year, Price, Mileage, Color, Image URL

Sync Status

Not configured

Instructions:

  1. Create an Excel file in SharePoint with your vehicle inventory
  2. Copy the SharePoint URL and Spreadsheet ID above
  3. Click "Setup SharePoint Sync" to establish the connection
  4. Your inventory will automatically sync with the Excel file
